# ButtonOptions

A ButtonOptions is used to present the different main options from the Home App 

Use ButtonOptions to create the main screen, to be able to redirect to the other parts of the site

<Canvas of={ButtonOptionsStories.Default} />

## Usage

The component comes in different color variants such as `primary`, `secondary`, `success`, `info`, etc. which you can use to alter the color of the button,
and different variants, such as `text`, `outlined` and `contained`, that affects the look and feel.
